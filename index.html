Github

Github est notre outil de travail principal en A2 ; que ce soit pour des travaux personnels, des travaux d'équipe le réflexe que je dois avoir, est la maîtrise du terminal et mettre à jour régulièrement nos repositories

Laragon

Laragon est le premier outil dev que j'ai utilisé en A2, il est très complet ; doté d'un terminal et d'une base de données. C'est aussi la première fois que j'ai utilisé un terminal.

SASS

Sass est le dernier langage de programmation que j'étudie : c'est une extension du css qui va permettre de mieux organiser ce dernier.

Tailwind

C'est la deuxième extension de css que j'ai expérimenté dans une intégration. Cela m'a permis d'être plus rigoureux sur l'optimisation de mon css.
